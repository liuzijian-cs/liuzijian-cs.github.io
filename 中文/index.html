    个人简述 / Biography

    嗨！我目前在上海大学计算机工程与科学学院攻读计算机科学硕士学位。我对生活充满热情，对计算机科学有着浓厚的兴趣。我的研究重点是时间序列预测、时空预测以及海洋大数据的预测和分析😊。

    经过一年的硕士学习，我对深度学习产生了深厚的兴趣，并逐渐掌握了各个领域的多样化技术栈。我关注最新的算法，包括KAN、mamba、diffusion以及BYOL、CLIP等对比学习方法和Flash Attention、FlowFormer等Transformer变体。在徐凌宇教授的指导和实验室同门的支持下，我专注于时空序列预测的创新研究。目前已完成多个研究项目，并正在解决时空预测中帧间变化大和误差累积的挑战。实验结果令人鼓舞，我正在进行进一步的实验并撰写相关论文。

    我对人工智能充满热情，渴望通过对人工智能的研究和应用，为人类社会发展贡献微薄的个人力量👍🌏。

    项目研究 / Projects:

    🔹[2024-07] 基于时空动态图神经网络的海洋温度预测预报技术
    该项目通过图神经网络将海平面抽象为图邻接矩阵，并随时间更新这些矩阵，打破了地理位置的限制。项目通过将海表温度和三维海洋温度数据转换为图结构，有效学习并预测了海洋数据中的复杂关系和动态变化。这项技术已在现实世界的数据集中广泛应用，证明了其在各种海洋环境下的适用性和准确性。

    🔹[2024-04] Academic Code Annotator, 基于LLAMA3的学术代码解读器.(Github 35🌟)
    Academic Code Annotator 是基于LLAMA 3的学术代码解读器，旨在通过全自动的代码注释，帮助用户快速理解与论文相关的编程代码。通过结合最新的LLAMA 3 8B instruct模型，本项目支持用户上传论文和代码，实现文献总结和注释代码的下载，极大地提升了研究效率和代码理解的深度。项目已部署在华为云上，并提供公网web交互界面及支持低算力的本地部署选项。

    🔹[2023-06] 基于树莓派小车的SLAM程序设计与开发
    此项目专注于组装一辆装备了四种先进激光SLAM技术的自动导引车（AGV），特色在于整合了ORB-SLAM3。这一尖端算法显著提升了车辆的自主导航和实时地图构建能力。成功的实施使得车辆能够在复杂环境中进行精确的环境映射和可靠的导航，凸显了车辆的操作效率得到了增强。
